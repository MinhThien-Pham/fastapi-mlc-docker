# KRSP_SecureLLM

Secure and fault-tolerant LLMs for embedded systems.

## 🚀 Quick Start

```bash
docker-compose up --build
```

Access the API at `http://localhost:8000`.

## 📌 Endpoints

- `GET /`: Welcome message
- `GET /health`: Service health status
- `POST /inference`: LLM inference (Integration pending)
